# Bitget AI Base Camp Hackathon S1 · GuardPilot

This repository contains the hackathon project under `guardpilot/`.

## Run from repository root

```bash
npm run replay
npm run dev
```

Then open:

- Dashboard: http://localhost:5173
- API docs: http://localhost:8000/docs
- Health check: http://localhost:8000/health

## Useful commands

```bash
npm run replay   # reproduce sample trading audit records
npm run dev      # start API + dashboard
npm run test     # run backend tests
npm run build    # build web dashboard
```

## Project

See the full project README:

- [`guardpilot/README.md`](guardpilot/README.md)

GuardPilot is a risk gateway and paper trading evaluation sandbox for autonomous trading agents, built for the Trading Infra track.
